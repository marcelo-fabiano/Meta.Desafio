# Desafio Meta
Desafio enviado pela Meta para testes de conhecimento.

### Projetos incluídos na sollução
#### Meta.Desafio.WebApi
Projeto do desafio de construção de uma api para gestão de contatos

#### Meta.Desafio.CRUD
Projeto de desafio de contrução de uma interface simples de CRUD (Create, Read, Update and Delete) de duas entidades simples

#### Meta.Desafio.Logica
Projeto de desafio de contrução de lógica de programação para algumas situaçãoes simples de algorítimo
